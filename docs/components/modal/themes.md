# Modal Themes

Modal Themes defines visual identity of your modal providing unique layout and feel for the modal.

## Available Themes

Dyvix provides a wide range of themes. You can trigger these by passing the string name or the exported constant object for types safety.

- `DYVIX_MODAL_THEME.SINGULARITY`: `'Singularity'`
- `DYVIX_MODAL_THEME.INDUSTRIAL`: `'Industrial'`
- `DYVIX_MODAL_THEME.EMBER`: `'Ember'`
- `DYVIX_MODAL_THEME.FROST`: `'Frost'`
- `DYVIX_MODAL_THEME.BLADE`: `'Blade'`
- `DYVIX_MODAL_THEME.NEON`: `'Neon'`
- `DYVIX_MODAL_THEME.AURORA`: `'Aurora'`
- `DYVIX_MODAL_THEME.SUNSET`: `'Sunset'`
- `DYVIX_MODAL_THEME.OCEAN`: `'Ocean'`
- `DYVIX_MODAL_THEME.FOREST`: `'Forest'`
